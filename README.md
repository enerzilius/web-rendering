# Rendering 3d objects using HTML Canvas

Based on: https://www.youtube.com/watch?v=qjWkNZ0SXfo

# How to run
- Just open `index.html` on your browser. Such as:
```
> firefox index.html
```
# How to use
- You can change the default line color, canvas background color and select a .obj file to be rendered.
